Before the workshop

We only have so much time to learn & practice Git in the workshop. Because of that, it would be very helpful if you could do a few things before you get to class:
  • Install Git on the computer you're bringing to class
  • Set up a GitHub account if you haven't already
  • Connect your workshop computer to GitHub by creating an SSH key, and then copying and pasting that key into your GitHub account settings.

If you have any questions, please contact us.
